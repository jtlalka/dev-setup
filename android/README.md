# Android Studio

## Custom shortcuts
- `⌘⌥⌃ + O` -- Open recent
- `⌘⌥⌃ + Q` -- Close project
- `⌘⌥⌃ + S` -- Sync Project with Gradle
- `⌘⌥⌃ + G` -- Build (G like Gradle)
- `⌘⌥⌃ + R` -- Rebuild project
- `⌘⌥⌃ + B` -- Branches...
- `⌘⌥⌃ + A` -- Annotation
- `⌘⌥⌃ + T` -- Terminal

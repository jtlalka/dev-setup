### Chrome

General:
```
cmd + shift + F              -- open/close full screen mode
cmd + L                      -- jump to address bar
```

Navigation:
```
cmd + T                      -- open new tab
cmd + W                      -- close curren tab
cmd + shift + T              -- open closed tab
cmd + [                      -- move to previous page
cmd + ]                      -- move to next page
cmd + shift + [              -- move to previous tab (left tab)
cmd + shift + ]              -- move to next tab (right tab)
```

View:
```
cmd + R                      -- reload current page
cmd + +                      -- incrise page content size
cmd + -                      -- decrise page cpmtent size
cmd + 0                      -- reset content size to default
```

### IntelliJ

General:
```
cmd + shift + A              -- show actions popup
alt + enter                  -- show context help popup
cmd + ,                      -- application preferences
cmd + ;                      -- project preferences
```

Navigation:
```
cmd + E                      -- show recent files
cmd + L                      -- move cursor to line
cmd + [                      -- move to previous currsor position
cmd + ]                      -- move to next currsor position
cmd + shift + [              -- move to previous file (left tab)
cmd + shift + ]              -- move to next file (right tab)
cmd + W                      -- close tab
shift + ESC                  -- close active window
cmd + U                      -- move to super method
cmd + B                      -- move to declaration
cmd + alt + B                -- move to implementation
cmd + O                      -- open class
cmd + shift + O              -- open file
cmd + F7                     -- show usage in find window
cmd + alt + F7               -- show usage in popup
ctrl + H                     -- open hierarchy
ctrl + J                     -- open quick documentation
F2                           -- move to next error/warning
F3                           -- toggle bookmark
cmd + F3                     -- show bookmarks
cmd + arrow up               -- show package path
cmd + F12                    -- show class structure
alt F1 -> 1                  -- show file in tree structure
alt F1 -> 8                  -- open file in file explorer
cmd + down                   -- move to selected file
alt + space                  -- show method implementation
ctrl + alt + up/down         -- next/previous highlight usages
ctrl + shift + up/down       -- next/previous method in class
cmd + plus                   -- expand all methods/files
cmd + minus                  -- collapse all methods/files
```

Coding:
```
cmd + N                      -- generate new code
cmd + J                      -- show live templates
cmd + P                      -- show parameters
cmd + D                      -- duplicate line
cmd + backspace              -- remove active line
ctrl + space                 -- auto completion
ctrl + shift + space         -- smart auto completion
cmd + shift + T              -- create test for class
cmd + shift + up/down        -- move block up or down
alt + shift + up/down        -- move line up or down
alt + shift + left/right     -- marked text left or right
cmd + shift + V              -- paste from clipboard history
```

Multiselection:
```
ctrl + G                     -- select next same phrase in file
cmd + G                      -- jump next same phrase in file
ctrl + cmd + G               -- select all same phrase in file
ctrl + shift + G             -- undo last selected phrease
alt + shift + click          -- add another cursor
alt -> alt + up/down         -- duplicate cursor to next/previous line 
```

Refactoring:
```
ctrl + T                     -- refactor menu
ctrl + O                     -- overade method
cmd + alt + T                -- surround with..
shift + F6                   -- rename property or class
cmd + alt + L                -- format code
cmd + alt + V                -- extract to variable
cmd + alt + F                -- extract to field
cmd + alt + M                -- extract to method
cmd + shift + U              -- replace to upper case
cmd + shift + F              -- find in path
cmd + shift + R              -- replace in path
```

Debugging:
```
ctrl + shift + R             -- run context configuration
ctrl + shift + D             -- debug context configuration
cmd + alt + R                -- run to next brake point
cmd + alt + F8               -- evaluate selected code
cmd + F8                     -- put brake point
F8                           -- step down
```

VCS:
```
cmd + K                      -- show git console
cmd + D                      -- show diff
F7                           -- next change (diff)
shift + F7                   -- previous change (diff)
cmd + alt + Z                -- undo uncommitted changes on selected area
```

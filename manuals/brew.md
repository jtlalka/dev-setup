### Brew

General:
```
brew list                    -- show list of all installed packages
brew list --cask             -- show list of native macOS installed packages
brew info <package>          -- show information about package
brew search <text>           -- search in all brew repositories
brew search --cask <text>    -- search in native macOS repositories
brew doctor                  -- validate brew packages system and environment
brew cleanup                 -- crear cache and outdated data
```

Packages:
```
brew install <package>       -- install selected package
brew uninstall <package>     -- uninstall selected package
brew update                  -- update brew repositories
brew upgrade                 -- upgrade all installed packages
brew upgrade <package>       -- update specific package
brew outdated                -- check for update packages
```

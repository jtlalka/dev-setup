# Android Studio

## Custom shortcuts
- `⌘⌥⌃ + O` -- Open recent
- `⌘⌥⌃ + Q` -- Close project
- `⌘⌥⌃ + S` -- Sync Project with Gradle
- `⌘⌥⌃ + G` -- Build (G like Gradle)
- `⌘⌥⌃ + D` -- Attach Debug
- `⌘⌥⌃ + R` -- Rebuild project
- `⌘⌥⌃ + B` -- Branches...
- `⌘⌥⌃ + A` -- Annotation
- `⌘⌥⌃ + T` -- Terminal
